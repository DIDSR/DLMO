# Sample size determination using pilot study

This script performs sample size determination for a split-plot design, as detailed in Section IV in the [supplementary material](https://arxiv.org/src/2602.22535v1/anc/DLMO_supp.pdf) of our [DLMO paper](https://arxiv.org/abs/2602.22535). To run the script, simply execute `sample_size_est_BDG.R`. To use your own pilot data, please replace `pilot_data.csv` with your data following the same format, and update proportion correct by DLMO and its variance in the `sample_size_est_BDG.R`.

# Additional resource
This folder contains the supporting document `sizing_2AFC.pdf`, which provides additional details on sample size estimation using study designs other than split-plot, such as fully crossed, flat without overlap, and flat with overlap, fixed number of readers.
