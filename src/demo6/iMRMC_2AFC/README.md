# iMRMC helper scripts

This folder contains supporting R scripts used by the statistical analysis examples in demo 6. These files provide helper functions and interfaces related to multi-reader multi-case and 2AFC analysis.

They are kept in the repository as reference material for the power-analysis and pivotal-study workflows. In normal use, you typically run the top-level scripts in `../power_analysis/` or `../pivotal_study/` rather than executing these helper files directly.
