# Sample size determination using pilot study

The scriptsample size determination in our paper. To run the script, simply execute `power_analysis_BDG.R`. To use your own pilot data, please replace `pilot_data.csv` with your data following the same format, and update proportion correct by DLMO and its variance in the `power_analysis_BDG.R`.

# Additional resource
This folder contains the supporting document `sizing_2AFC.pdf`, which provides additional detail for the sample size determination analysis used in our paper.

Use this document together with the runnable script in `../power_analysis/` when adapting the statistical analysis to a new study.
